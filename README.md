# Modeling and Control of Host–Parasitoid Dynamics in Sugarcane Borer Biological Management

This repository contains the code used in the simulation, optimization, and analysis of the model under the title above.

## Abstract

The objective of this work is to utilize biological control in a system involving the sugarcane borer and its parasitoids, the egg parasitoid _Trichogramma galloi_ and the larval parasitoid _Cotesia flavipes_. The population dynamics are modeled by a host-parasitoid system considering six ordinary differential equations. A Type II functional response is used to describe the interactions between the hosts and parasitoids. Biological control strategies for pest management are proposed, using continuous control and impulsive control methodologies. Numerical simulations are conducted and discussed. The results are compared among the different pest management strategies and are also qualitatively compared with experimental data presented in the specialized literature.


*Keywords:* host-parasitoid system, sugarcane borer, nonlinear dynamics, economic damage threshold, biological control, optimal control, impulsive control, feedback stabilization.
